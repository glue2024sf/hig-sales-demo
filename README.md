# HIG社 営業 デモ画面（レビュー用）

HIG社（DHCブランドのホテルアメニティ販売）向け Salesforce 構築の、Lightning 風 静的HTMLデモです。**外部依存なし**で、`index.html` をブラウザで開けば全ページを回遊できます。

## 前提（HIG社の業態）
- チャネル軸: 直販 / 東京マツシマ / JTB商事 / ニシカワヤ / 特販
- 顧客: チェーン運営会社（TOP147）⇄ 個別施設の二層（顧客コード5桁）
- 商品: 品番体系（A〇G/A〇M/A〇X/B/C/D/SG/SM/SJ）
- 出荷: Tシステム（Excelベース受発注）連携、DHCベース(1-12月)/HIGベース(4-3月)
- 営業ステージ: 情報未収集→情報収集済→営業済→内定→受注→継続発注→ロスト

## 開き方
1. このフォルダを丸ごと保存
2. `index.html` をダブルクリック（任意のブラウザ）
3. 上部タブ＋ホームのリンクから各画面へ移動

## 画面一覧（業務シーン網羅版）
- ホーム
- ダッシュボード: 週報 / 営業チーム / 個人 / 四半期・経営 / 出荷・売上(Tシステム)
- **A. 顧客・名寄せ**: チェーン一覧 / チェーン詳細 / 施設詳細 / 取引先責任者 / 新規登録 / 対応表整備(名寄せ)
- **B. 営業プロセス**: 商談一覧 / 商談詳細 / 見積・提案 / 開業案件 / ロスト管理 / 承認プロセス
- **C. 活動・追客**: 活動一覧・タイムライン / 活動入力 / ベルセールス議事録 / ToDo・期限切れアラート
- その他: サンプル管理 / 商品（品番）/ 週報入力
