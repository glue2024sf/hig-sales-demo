# HIG 営業 デモ画面（レビュー用）

Salesforce Lightning 風の静的HTMLデモです。**外部依存なし**で、`index.html` をブラウザで開けば全ページを回遊できます。

## 開き方
1. このフォルダを丸ごと保存
2. `index.html` をダブルクリック（任意のブラウザ）
3. 上部タブ＋ホームのリンクから各画面へ移動

## 画面一覧
- ホーム（`index.html`）
- ダッシュボード: 全体MTG / チーム / 個人 / 経営 / 出荷
- 取引先（一覧・詳細）/ 商談（提案型・補充型）/ 活動入力 / 週次レポート / 商品 / 追客

## フィードバック方法
各画面のスクリーンショットに**赤字でコメント**をお願いします（前回と同じ方式）。
各ページ下部の 💡 が論点・確認したい点です。

> ⚠️ 数値・社名はすべてダミーです。
